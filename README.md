# PC Parts Stock Control System

A Java-based stock control system with a graphical user interface for managing PC component inventory. Developed as part of MSc Computer Science coursework, focusing on object-oriented programming, inventory management and GUI-based application design.

## Technologies

- Java
- Java Swing / JavaFX (whichever you used)
- Object-Oriented Programming
- Eclipse

## Features

- GUI-based inventory management system
- Add, update and remove PC component stock
- Search and filter inventory records
- Dynamic stock tracking functionality
- Structured object-oriented class design
- Input validation and error handling

## Concepts Demonstrated

- Encapsulation
- Inheritance
- Abstraction
- Event-driven programming
- GUI development
- Inventory management logic
